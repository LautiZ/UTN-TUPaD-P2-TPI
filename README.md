## Tema elegido: Usuario → CredencialAcceso

El dominio seleccionado fue Usuario → CredencialAcceso, una relación que representa un caso muy común en aplicaciones reales: cada usuario registrado posee exactamente una credencial de acceso asociada, que contiene información sensible como la contraseña, su hash y el estado de la cuenta.

## Descarga de repositorio

Clonar el repositorio:

```bash
  git clone https://github.com/LautiZ/UTN-TUPaD-P2-TPI
```

Ingresar a la carpeta del programa:

```bash
  cd UTN-TUPaD-P2-TPI
```

## Ejecucion

Para poder usar correctamente este proyecto primero debe ejecutar los siguiente archivos. Para ejecutarlos simplemente busque el archivo y haga click derecho y Run file.

`DatabaseGenerator.java`
: Este archivo genera la base de datos para poder utilizar la aplicacion correctamente.

(Opcional)
`DatabaseSeeder.java`
: Este archivo popula la base de datos para tener datos de ejemplo y poder probar algunas funciones mas comodamente.

Una vez ejecutados los arhivos anteriores, es momento de pasar al nucleo de nuestro programa que es el archivo llamdo `Tpi.java`. En este encontraran la funcion main con el menu de opciones, es simplemente ejecutarlo y seguir las opciones que se muestran por consola.
